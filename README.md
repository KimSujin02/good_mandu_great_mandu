# 🥟🥟굿 만두, 그레이트 만두🥟🥟

파이썬 콘솔에서 실행되는 **미니 게임**입니다.  
재료 선택 → 다지기 → 찜기 → 심사까지,  
직접 키보드를 두드리며 만두를 완성하고 점수를 겨뤄보세요!



https://github.com/user-attachments/assets/61c11d06-99ba-46ee-85f2-bba411062678



---

## 🎮 게임 개요

- **플랫폼**: Windows 콘솔 (Command Prompt / PowerShell / VS Code 터미널)
- **입력 방식**: 키보드 입력
- **플레이 시간**: 약 1~2분
- **특징**
  - 모드 선택 (이지 / 하드)
  - 실시간 키 입력으로 점수 계산
  - 랜덤 요소가 포함된 찜기 단계
  - 누적 랭킹 시스템

---

## 🧩 게임 진행 단계

### 1️⃣ 모드 선택
- **e (Easy)**  
  - 재료 선택 → 다지기 → 찜기 → 심사
- **h (Hard)**  
  - 재료 선택 → 양념 선택 → 다지기 → 찜기 → 심사

---

### 2️⃣ 재료 선택 단계
- 제시된 재료 목록 중 **5개를 직접 입력**
- 각 재료마다 점수가 다름
  - 좋은 재료: 가산점
  - 이상한 재료: 감점

**평가 기준**
- 최고: 40점 이상
- 굿: 20 ~ 39점
- 최악: 19점 이하

---

### 3️⃣ (하드모드 전용) 양념 선택
- 양념 재료 중 **5개 선택**
- 재료 단계와 동일한 방식으로 점수 계산

**평가 기준**
- 최고: 25점 이상
- 굿: 20 ~ 24점
- 최악: 19점 이하

---

### 4️⃣ 다지기 단계 🔪
- **3초 동안 키보드를 두드려주세요!**
- 입력 횟수에 따라 점수 부여됩니다.

**점수 기준**
- 80회 이상: +100점 (완벽!)
- 50 ~ 79회: +50점
- 49회 이하: -10점

⚠ **영문 키보드 상태에서 진행해야 정확합니다.**

---

### 5️⃣ 찜기 단계 🔥
- 찜기 가동 중 랜덤 이벤트 발생
- **-20 ~ +100점** 사이의 보너스(또는 패널티) 점수 획득

---

### 6️⃣ 심사 단계 👨‍⚖️
- 모든 단계 점수를 합산하여 최종 평가 진행
**[!!!심사위원은 랜덤 등장!!!]**

**모드별 판정 기준**
- 이지 모드
  - 최고: 120점 이상
  - 굿: 70 ~ 119점
- 하드 모드
  - 최고: 150점 이상
  - 굿: 100 ~ 149점

---

## 🏆 랭킹 시스템

- 게임 종료 시 플레이어 이름과 점수가 랭킹에 저장됨
- 상위 5명까지 출력
- 여러 판 플레이 가능

---
